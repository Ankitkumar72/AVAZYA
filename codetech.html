Code tech

Code Tech is a thrilling event that will take place at the Avazya techfest, which consists of various rounds of quiz, debugging, and coding challenges. The event is designed to test the coding and technical skills of participants and challenge them to think critically and innovatively.

3 Rounds:

Round 1 : The quiz round, participants will face questions related to coding, programming, and other technical topics. The round will test their knowledge of programming concepts and problem-solving skills.

Round 2 : The debugging round, participants will be presented with a piece of code with errors, and they will have to identify and fix the errors in the shortest possible time. This round is designed to test their debugging skills, ability to identify errors and bugs in code, and troubleshoot issues.

Round 3 : The coding challenge round, participants will be given a task or a problem to solve, and they will have to use their programming skills to develop a solution. The round will test their ability to think creatively, come up with innovative solutions, and code efficiently.

Code Tech is open to students from all disciplines, and it provides a fantastic opportunity for participants to showcase their technical skills, learn from their peers, and network with industry experts. We can't wait to see the participants showcase their coding and technical skills in this year's Avazya techfest!

Rules :

1. Participants are not allowed to discuss the problems with members of other teams.
2. Participants are not allowed to use non-contest software on their computers/laptops.
3. Participants are not allowed to use web browsers, telnet, ssh, ftp, sftp or other network software to obtain help on their contest problems.
4. The efficiency of the program/algorithm are based on their execution time.
5. Your programs must not do any file input or output. All contest input will come from the keyboard, and all output must go to the console.
6. Participants are not allowed to use smart watches, mobile phones or calculators in the contest.
7. The judges’ decision is final.
